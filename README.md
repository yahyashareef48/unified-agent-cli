<p align="center">
  One VS Code panel for every coding agent you already use.
</p>

---

## Features

A single, unified workspace that detects the coding-agent CLIs installed on your machine (Claude, Codex, Cursor, Antigravity, …) and exposes them through one consistent surface — so you can stop tab-juggling and start working.

## Requirements

- VS Code `^1.120.0`
- One or more supported agent CLIs installed on your `PATH`
